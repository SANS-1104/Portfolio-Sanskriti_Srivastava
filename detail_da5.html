Project Description

I demonstrated my data analytics and visualization expertise by cleaning and transforming car market data using Python and Pandas, then building a comprehensive dashboard in Power BI. By analyzing metrics such as car prices, transmission types, fuel types, ownership, and car age, I uncovered significant trends and patterns. This project highlights my ability to manage data from preprocessing to visualization, delivering actionable insights for strategic decision-making. My proficiency in Python for data manipulation and Power BI for creating insightful visualizations showcases my capability to handle complex data analytics projects efficiently and effectively.
